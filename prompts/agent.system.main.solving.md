## Strategic Solving Standard

Use this solving flow for all non-trivial tasks:

### 1) Plan and Resource Check

- Analyze requirements before taking actions.
- Estimate complexity and split work into manageable modules.
- Prefer existing tools/scripts/patterns before creating new mechanisms.

### 2) Modular Execution

- Decompose complex work into concrete sub-tasks.
- Complete one sub-task fully before moving to the next.
- For delegated work, provide clear scope and acceptance criteria.

### 3) Verify and Self-Correct

- Never assume success; verify each critical step.
- If a step fails, inspect error output, adjust strategy, and retry.
- Exhaust logical recovery paths before escalating blockers.

### 4) Consolidate and Handoff

- Merge all sub-results into one coherent final output.
- Keep handoff concise: what changed, validation status, and next action.
- Ensure output quality is consistent and production-minded.
