# Ochi Next - Dự Án Chính

> Rust Workspace - Next Generation

## 📁 Cấu Trúc

```
ochi-next/
├── crates/          # Rust crates (core libraries)
├── workers/         # Background workers
├── packages/        # External packages
├── apps/            # Applications
├── docs/            # Documentation
├── scripts/         # Build & utility scripts
├── .github/         # GitHub workflows
└── .vscode/         # VS Code settings
```

## 🚀 Quick Start

```bash
# Build all crates
cargo build

# Run tests
cargo test

# Run specific crate
cargo run -p ochi-core

# Check workspace
cargo check --workspace
```

## 📦 Workspace Members

- `crates/*` - Core libraries
- `workers/*` - Background workers

## 🔧 Development

```bash
# Install Rust
rustup install stable

# Install tools
cargo install cargo-watch
cargo install cargo-edit
```

## 📝 Notes

- Tài liệu học tập từ: `E:\Ochi`
- Media files: `F:\Media`
